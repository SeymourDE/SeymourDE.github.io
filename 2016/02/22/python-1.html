变量

变量在程序中就是用一个变量名表示，变量名必须是大小写英文、数字和_的组合，且不能用数字开头，比如：a = 1

常量

所谓常量就是不能变得变量，比如常用的数学常数π就是一个常量，通常用全部大写的变量名表示常量，比如：PI = 3.1415926

基础类型

  • 字符串
  • 布尔类型
  • 整数
  • 浮点数
  • 数字
  • 列表(list)
  • 元组(tuple)
  • 字典(dict)
  • 日期

整数

  • python 可以处理任意大小的整数，当然包括负整数，在程序中的表示方法和数学上的写法一模一样，例如: 1, 100, -10等。
  • 也可以用进制表示，二进制:0b1000、八进制:0010、十六进制: 0x1f。

浮点数

  • 浮点数数也就是小数，例如，1.32, 23.1, -10.11等等，但是对于很大或很小的浮点数，就必须用科学计数法表示，把10用e替代，1.23X10的9次方，就是1.23e9

字符串

  • 使用单引号，例如: ‘string’。
  • 使用双引号，例如: “string”。
  • 使用三引号，表示多行字符串，可以在三引号中自由使用单引号和双引号。
  • 转义字符\可以转义很多字符，比如\n 表示换行，\t表示制表符，字符\本身也要转义，所以\表示的就是字符\。

布尔值

  • 布尔值只有 True和False两种值(请注意大小写)。
  • 可用于运算and, or, not中。

空值

  • 空值是pythone里一种特殊的值，用None表示，None不能理解为0，因为0是有意义的，而None是一个特殊的空值。
