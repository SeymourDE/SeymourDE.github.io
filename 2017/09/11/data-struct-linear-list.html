线性结构(3)

Sep 11 2017

线性表

线性表(linear list)是一种典型数据结构，比如说26个英文字母(a, b, c, …, x, y, z)的排列，还有PHP底层的双向链表等等。

线性表简称表，是零个或多个数据元素组成的有限序列(a0, a1, …, ai, ai+1, …, an)。在复杂的线性表中，一个数据元素可以由若干个数据项组成。

首先看一下线性结构的二元组 B=(K, R), K={a0, a1, …, an-1}, R={r}。

对于以上线性结构，当n=0时，线性表为空，我们称之为空表。

对于n>0时，对于内部任何数据元素都存在一种关系< ai, ai+1 >, 我们说ai是ai+1的前驱，ai+1是ai的后继。

线性结构的，

  • 有且只有唯一的开始节点(表头), 它没有前驱，有一个唯一的直接后继。
  • 有且只有唯一的终止节点(表尾), 他有唯一一个直接前驱，没有后继。
  • 其他节点称为内部节点，每一个内部节点都有且仅有一个唯一的直接前驱，也有唯一一个直接后继。
  • 前驱/后继关系，具有反对称性和传递性。

线性结构分类

1. 线性表

线性表是所有表目都是同一类型结点的线性表，不限制操作形式，根据存储的不同分为: 顺序表和链表。

顺序表, 是一组地址连续的存储单元依次存储线性表的数据元素。

链表, 是由一系列不必在内存中项链的结构组成，每一个结构均含有表元素和指向包含改元素后继元素的结构的指针, 分为，单链表、双链表、循环链表。

链表与数组的区别:

  • 数组静态分配内存(数组元素在栈区)，链表动态分配内存(链表元素在堆区)。
  • 数组在内存中连续，链表是不连续的。
  • 数组利用下标定位，时间复杂度为O(1), 链表定位元素时间复杂度O(n)。
  • 数组插入或删除元素的时间复杂度O(n), 链表的时间复杂度为O(1)。

2. 栈

栈的插入和删除操作都限制在表的同一端进行。

3. 队列

队列插入操作在表的一端，删除操作在另一端。
